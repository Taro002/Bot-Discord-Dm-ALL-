# Discord Mass DM Bot

A simple Discord.js bot that allows server administrators to send a direct message to all server members with a confirmation step before sending.

## Features

- Message preview before sending
- Confirmation / cancellation via buttons
- Sends DMs only to non-bot members
- Progress tracking (sent / failed)
- Configurable delay to avoid rate limits

## Requirements

- Node.js v18 or higher
- A Discord bot token
- Discord.js v14

## Installation

1. Clone the repository or copy the source code
2. Install dependencies:

```bash npm install discord.js```
